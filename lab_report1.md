Basic filesystem commands
==

Three basic commands:
--
1. 'ls'
2. 'cd'
3. 'cat'

'ls'
--
1. with no arguments
   'ls'
2. with a path to a directory as an argument
   'ls messages'
3. with a path to a file as an argument
    'ls Hello.java'

'cd'
--
1. with no arguments
   'cd'
2. with a path to a directory as an argument
   'cd messages'
3. with a path to a file as an argument
   'cd Hello.java'

'cat'
--
1. with no arguments
   'cat'
2. with a path to a directory as an argument
   'cat messages'
3. with a path to a file as an argument
   'cat Hello.java'
